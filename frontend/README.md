Untuk Menjalankan Aplikasi ini


---------- Running Backend ---------------

Install modul di requirements.txt

---------- Running Frontend ---------------
Masuk ke folder frontend
Install modul (npm install)
--- untuk development
Jalankan perintah (npm run dev)

--- Untuk Produksi
Jalankan perintah (npm run build
